# iVolunteer

iVolunteer is a web-based platform that connects **volunteers** and **organizations** to make it easier to find, manage, and participate in volunteer opportunities.

Whether you are looking to volunteer or manage an organization, this system helps you get started quickly and stay organized.

---

## What You Can Do in iVolunteer

### For Volunteers
- Create an account and log in easily (or use Google login)
- Browse available volunteer opportunities
- Apply to opportunities with one click
- Track your application status (Pending, Approved, Rejected)
- View your activity and participation history
- Receive notifications about updates from organizations

---

### For Organizations
- Create and manage volunteer opportunities
- Review applications from volunteers
- Approve or reject applicants
- Manage your organization profile
- Track volunteers participating in your programs

---

### For Administrators
- Manage all users (volunteers and organizations)
- Monitor system activity
- Handle reports and issues
- Control platform-wide data and content

---

## Logging In

You can log in using:

### Option 1: Email Login
- Register using your email and password
- Log in directly from the login page

### Option 2: Google Login (Recommended)
- Click **“Continue with Google”**
- You will be redirected to Google for authentication
- After approval, you will be logged in automatically


---

## How to Use the System

### Step 1: Sign Up or Log In
Create an account or use Google login.

### Step 2: Choose Your Role
- Volunteer → Apply for opportunities
- Organization → Post opportunities

### Step 3: Start Using the Platform

#### If you are a Volunteer:
1. Go to **Opportunities**
2. Click an opportunity you like
3. Click **Apply**
4. Wait for approval from the organization
5. Track your application status anytime

#### If you are an Organization:
1. Go to **Dashboard**
2. Create a new opportunity
3. Wait for volunteers to apply
4. Review applications
5. Approve or reject applicants

---

## Notifications
You will receive notifications for:
- Application updates
- New responses from organizations
- System announcements

You can also mark all notifications as read anytime.

## API Access (For Developers)
This system also provides an API for integration.

Examples:
- `/api/opportunities/` — list volunteer opportunities
- `/api/applications/` — view your applications
- `/api/organizations/` — list organizations
- `/api/posts/` — community posts

> API access requires login.
